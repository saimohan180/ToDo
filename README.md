# TaskFlow

A modern, local-first task management application.

## 🚀 Installation & Setup

1. **Clone the repository:**
   ```bash
   git clone https://github.com/saimohan180/ToDo.git
   ```

2. **Navigate to the project directory:**
   ```bash
   cd ToDo
   ```

3. **Install dependencies:**
   ```bash
   npm install
   ```

4. **Run in development mode:**
   ```bash
   npm run dev
   ```
   Access the web interface at: [http://localhost:5173](http://localhost:5173)

5. **Build and run for production:**
   ```bash
   npm run build
   ```
   Access the production server at: [http://localhost:4010](http://localhost:4010)

## 🔐 Secret Journal
